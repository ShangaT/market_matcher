Market Matcher

Сравнение продуктов питания из разных магазинов


Проект Market Matcher предназначен для облегчения поиска продуктов питания путем сравнения цен из разных магазинов.
Проект стремится предоставить пользователям инструмент для принятия информационных решений при покупке продуктов.
